# Personal Resume Website

## Overview
This project is a personal resume and portfolio website built to showcase my education, technical skills, and professional experience as a Computer Science student. The site is designed to be simple, clean, and easy to navigate, serving as an online version of my resume.

## Technologies Used
- JavaScript
- HTML
- CSS

## Features
- Resume-style layout with sections for experience, education, and skills
- Interactive and responsive design
- Clean, readable UI optimized for desktop and mobile
- Organized timeline for professional experience

## Purpose
This project was created to:
- Practice front-end web development fundamentals
- Present my resume in a more interactive and accessible format
- Maintain a professional portfolio for internships and full-time roles

## How to Run
1. Clone or download the repository
2. Open `index.html` in a web browser
3. View and navigate the resume website locally

## Notes
- This is a personal project and will be updated as my experience grows
- The focus is on clarity, structure, and maintainable code rather than complex frameworks
